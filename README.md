# Reminder App

Reminder app made with React, Redux, tailwindCSS, typescript and more.
Deployed on [Netlify](https://www.netlify.com/)

Live preview  [reminder-app-vite](https://reminder-app-vite.netlify.app/)

# Installation
Install Dependencies 
```
npm i 
```
# Run
For developers
```
npm run dev
```

# Build 
```
npm run build
```
